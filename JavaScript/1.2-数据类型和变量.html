Number

123; // 整数123
0.456; // 浮点数0.456
1.2345e3; // 科学计数法表示1.2345x1000，等同于1234.5
-99; // 负数
NaN; // NaN表示Not a Number，当无法计算结果时用NaN表示
Infinity; // Infinity表示无限大，当数值超过了JavaScript的Number所能表示的最大值时，就表示为Infinity

可直接用四则运算
    1 + 2; // 3
    (1 + 2) * 5 / 2; // 7.5
    2 / 0; // Infinity
    0 / 0; // NaN
    10 % 3; // 1     注意%是求余运算。
    10.5 % 3; // 1.5

字符串

单引号或者双引号括起来的文本，比如“abc”，‘abc’。只有三个字符，单双引号都不算

布尔值

true  false

true; // 这是一个true值
false; // 这是一个false值
2 > 1; // 这是一个true值
2 >= 3; // 这是一个false值

&&  与运算   所有都为true，结果才为true
    true && true; // 这个&&语句计算结果为true
    true && false; // 这个&&语句计算结果为false
    false && true && false; // 这个&&语句计算结果为false

||  或运算    只要一个为true，结果就为true
    false || false; // 这个||语句计算结果为false
    true || false; // 这个||语句计算结果为true
    false || true || false; // 这个||语句计算结果为true

！  非运算    把true变为false，false变为true
    ! true; // 结果为false
    ! false; // 结果为true
    ! (2 > 5); // 结果为true

布尔值常用判断中
        var age = 15;
    if (age >= 18) {
        alert('adult');
    } else {
        alert('teenager');
    }

比较运算符
    2 > 5; // false
    5 >= 2; // true
    7 == 7; // true

==  ===
    false == 0; // true
    false === 0; // false

    NaN === NaN; // false
    唯一能判断NaN的方法是通过isNaN()函数：
    isNaN(NaN); // true

浮点数的相等比较
    1 / 3 === (1 - 2 / 3); // false
    Math.abs(1 / 3 - (1 - 2 / 3)) < 0.0000001; // true

null和undefined

null 表示空值，与0以及空字符串''不同，0是一个数值，''表示长度为0的字符串，而null表示空
undefined 表示未定义

数组 [ ]

数组是一组按顺序排列的集合，集合的每个值称为元素。JavaScript的数组可以包括任意数据类型:
    [1, 2, 3.14, 'Hello', null, true];
    new Array(1, 2, 3); // 创建了数组[1, 2, 3]

数组的元素可以通过索引来访问。请注意，索引的起始值为0：
    var arr = [1, 2, 3.14, 'Hello', null, true];
    arr[0]; // 返回索引为0的元素，即1
    arr[5]; // 返回索引为5的元素，即true
    arr[6]; // 索引超出了范围，返回undefined

对象 { }

        tags: ['js', 'web', 'mobile'],
        city: 'Beijing',
        hasCar: true,
